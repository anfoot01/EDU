Module 4. Operators

Theory:

  • Arithmetic: + - * / %
  • Comparison: == != > < >= <=

#include 
using namespace std;

    int x = 15;
    cout << (x > 10 && x < 20) << endl; // 1 (true)

}

Module 5. Conditionals

Theory:C++ uses if, else if, else , and switch .

using namespace std;

    int num = -5;
    if (num > 0) cout << "Positive";
    else if (num < 0) cout << "Negative";
    else cout << "Zero";
}

Module 11. Pointers and References

Theory:

Module 12. File I/O

Theory:

  • Use for file operations.
  • ofstream for writing, ifstream for reading.
#include 
#include 
using namespace std;

}
