Module 1. Introduction to Go

Theory: Go is a compiled, statically typed language created by Google. It is fast, simple, and great for concurrency and scalable systems.

package main
import "fmt"

func main() {
    fmt.Println("Hello, Go!")
}

Practice: Install Go and print "Hello, Go!".

Module 2. Variables and Data Types

Theory:

  • Declare with var or := .
  • Types: int, float64, string, bool .
  • Zero values: numbers → 0, strings → "", bool → false.
package main
import "fmt"

func main() {
    var age int = 21
    name := "Alice"
    isStudent := true

    fmt.Println(name, age, isStudent)
}

Practice: Create variables: name (string), age (int), isStudent (bool). Print them.

Quiz: Which operator allows short declaration?

Module 3. Constants

Theory: Use const for values that don’t change. Constants are immutable.

package main
import "fmt"

func main() {
    const Pi = 3.14159
    fmt.Println("Pi =", Pi)
}

Practice: Create a constant for Pi and print it.

Quiz: Which keyword defines constants?

Module 4. Operators

Theory:

  • Arithmetic: + - * / %
  • Comparison: == != > < >= <=
  • Logical: && || ! 
package main
import "fmt"

func main() {
    x := 15
    fmt.Println(x > 10 && x < 20) // true
    fmt.Println(10 % 3)           // 1
}

Practice: Check if a number is between 10 and 20.

Quiz: What does % do?

Module 5. Conditionals

Theory: Go uses if, else if , and else . No parentheses needed. 

package main
import "fmt"

func main() {
    num := -3
    if num > 0 {
        fmt.Println("Positive")
    } else if num < 0 {
        fmt.Println("Negative")
    } else {
        fmt.Println("Zero")
    }
}

Practice: Check if a number is positive, negative, or zero.

Quiz: Does Go require parentheses around conditions?

Module 6. Loops

Theory: Go has only one loop: for .

Forms: classic for range-based for , infinite loop.

package main
import "fmt"

func main() {
    for i := 1; i <= 10; i++ {
        fmt.Println(i)
    }
}

Practice: Print numbers from 1 to 10.

Quiz: Which loop type does Go have?

Module 7. Functions

Theory: Functions are defined with func . They can return multiple values.

package main
import "fmt"

func add(a int, b int) int {
    return a + b
}

func main() {
    fmt.Println(add(3, 5)) // 8
}

Practice: Write a function to add two numbers.

Quiz: Write a function to add two numbers.

Module 8. Arrays and Slices

Theory:

  • Arrays: fixed length.
  • Slices: dynamic, more common in Go.
package main
import "fmt"

func main() {
    fruits := []string{"apple", "banana", "orange"}
    fmt.Println(fruits[0])  // apple
    fmt.Println(len(fruits)) // 3
}

Practice: Create a slice of fruits and print the first one.

Quiz: Which is more common in Go: arrays or slices?

Module 9. Maps

Theory: Maps store key-value pairs. Keys must be unique.

package main
import "fmt"

func main() {
    capitals := map[string]string{
        "USA": "Washington",
        "UK":  "London",
    }
    fmt.Println(capitals["USA"])
}

Practice: Create a map of countries and capitals.

Quiz: What Go type stores key-value pairs?

Module 10. Structs

Theory: Structs group related fields. They are similar to classes without methods.

package main
import "fmt"

type Person struct {
    name string
    age  int
}

func main() {
    p := Person{name: "Alice", age: 21}
    fmt.Println(p.name, p.age)
}

Practice: Create a struct Person with name and age .

Quiz: Which keyword defines a struct?

Module 11. Pointers

Theory: Pointers hold memory addresses. Use & to get address, * to dereference.

package main
import "fmt"

func main() {
    x := 10
    p := &x
    *p = 20
    fmt.Println(x) // 20
}

Practice: Create a pointer to an integer and change its value.

Quiz: Which symbol gets the address of a variable?

Module 12. Concurrency with Goroutines

Theory: Goroutines allow concurrent execution. Use go keyword before a function. 

package main
import (
    "fmt"
    "time"
)

func say(msg string) {
    for i := 0; i < 3; i++ {
        fmt.Println(msg)
        time.Sleep(time.Millisecond * 500)
    }
}

func main() {
    go say("Hello")
    say("World")
}

Practice: Run two functions concurrently.

Quiz: Which keyword starts a goroutine?
