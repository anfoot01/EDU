Course "JavaScript from Scratch"

Module 1. Introduction

Theory: JavaScript is a programming language that makes websites interactive. It works inside browsers and is one of the core technologies of the web, together with HTML and CSS.

With JavaScript, you can:

  • Validate user input in forms.
  • Create interactive animations.
  • Build web apps and even mobile apps.

<script>
  alert("Hello, world!");
</script>

Practice: Make your own alert with the message "I am learning JS!".

Quiz: What is JavaScript mainly used for?

Module 2. Variables and Data Types

Theory: Variables store information in memory. There are three main keywords for declaring them: let, const, and var (old, rarely used). Data types include:

  • string – text data ("Hello")
  • number – numeric values (10, 3.14)
  • boolean – true or false
  • null – empty value
  • undefined – variable declared but not assigned
  • object – collection of key-value pairs
  • array – ordered list of values
let name = "Anna";
const age = 20;
let isStudent = true;

Practice: Create variables with your name, age, and student status. Log them with console.log().

Quiz: Which keyword is used for a constant variable?

Module 3. Operators

Theory: Operators allow you to perform actions with values. The most common are:

  • Arithmetic: +, -, *, /, %
  • Comparison: ==, ===, !=, <, >
  • Logical: &&, ||, !
let a = 5, b = 3;
console.log(a + b); // 8
console.log(a > b); // true

Practice: Calculate the average of three numbers.

Quiz: What is the result of 5 == "5"?

Module 4. Conditions

Theory: Conditions let your program make decisions. You can use if, else if, else, and switch. 

let age = 18;
if (age >= 18) {
  console.log("Access granted");
} else {
  console.log("Access denied");
}

Practice: Check if a number is even.

Quiz: What does the % operator do?

Module 5. Loops

Theory: Loops let you repeat actions. The main types are:

  • for – repeats for a set number of times.
  • while – repeats while a condition is true.
  • do...while – executes at least once, then repeats while condition is true.
for (let i = 1; i <= 5; i++) {
  console.log("Step", i);
}

Practice: Print numbers from 1 to 20.

Quiz: Which loop executes at least once even if the condition is false?

Module 6. Functions

Theory: Functions let you group reusable code. They can take parameters and return values.

function greet(name) {
  return "Hello, " + name + "!";
}
console.log(greet("Olivia"));

Practice: Write a function that calculates the square of a number.

Quiz: Which keyword is used to return a value from a function?

Module 7. Arrays

Theory: Arrays store multiple values in a single variable. You can use methods like push, pop, length, and forEach. 

let fruits = ["apple", "banana"];
fruits.push("orange");
console.log(fruits.length); // 3

Practice: Create an array of your favorite movies and print the first and last one.

Quiz: How do you get the length of an array?

Module 8. Events

Theory: Events let you react to user actions, like clicking, typing, or submitting forms.

<button onclick="sayHello()">Click me</button>
<script>
function sayHello() {
  alert("Hello!");
}
</script>

Practice: Make a button that changes the text on the page when clicked.

Quiz: Which event is triggered by clicking a button?
