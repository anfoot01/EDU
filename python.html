Module 1. Introduction to Python

Theory: Python is an interpreted, high-level, dynamically typed programming language. It is widely used in web development, data science, machine learning, automation, and scripting. Its syntax is simple and human-readable.

Module 2. Variables and Data Types

Theory:

  • Python is dynamically typed.
  • Data types: int, float, str, bool, list, tuple, set, dict.
  • Use type() to check a variable's type.
name = "Alice"
age = 21
is_student = True

print(name, age, is_student)
print(type(name), type(age), type(is_student))

Module 4. Conditional Statements

Theory: Python uses if, elif and else for decision making.

n = -5
if n > 0:
    print("Positive")
elif n < 0:
    print("Negative")
else:
    print("Zero")

Module 5. Loops

Theory: Loops let you repeat actions. The main types are:

  • while repeats while condition is true
  • break stops loop, continue skips iteration

  console.log("Step", i);
}

Module 11. OOP Basics

Theory: Python is object-oriented: classes define objects with attributes and methods.

class Car:
    def __init__(self, brand, speed):
        self.brand = brand
        self.speed = speed
    
    def drive(self):
        print(f"{self.brand} is driving at {self.speed} km/h")

car = Car("BMW", 120)
car.drive()

Practice: Create a Car class.

Quiz: Which keyword defines a class?
