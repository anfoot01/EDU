Course "HTML & CSS"

Module 1. Introduction to HTML & CSS

Theory: HTML (HyperText Markup Language) defines the structure of a web page, while CSS (Cascading Style Sheets) controls its design and layout. Together, they form the foundation of web development.

<!DOCTYPE html>
<html>
  <head>
    <title>Hello World</title>
  </head>
  <body>
    <h1>Hello, HTML!</h1>
  </body>
</html>

Practice: Create a basic HTML page that displays “Hello, HTML!”.

Quiz: What does HTML stand for?

Module 2. HTML Structure & Elements

Theory: HTML uses tags to define elements. Common ones include headings, paragraphs, links, and images.

<h1>Welcome</h1>
<p>This is a paragraph.</p>
<a href="https://learncodee.netlify.app">Visit LearnCodee</a>
<img src="image.jpg" alt="Example">

Practice: Add a heading, a paragraph, and an image to your page.

Quiz: Which tag is used for links?

Module 3. CSS Basics

Theory: CSS adds style to HTML elements. It can be written inline, internally (in <style>), or in external files (.css).

h1 {
  color: blue;
  font-family: Arial;
}

Practice: Change your heading color and font.

Quiz: Which property changes text color?

Module 4. The Box Model

Theory: Every HTML element is a box with content, padding, border, and margin.

div {
  width: 200px;
  padding: 10px;
  border: 2px solid black;
  margin: 20px;
}

Practice: Create a styled box with padding and a border.

Quiz: Which CSS property controls the space inside the element border?

Module 5. Layout with Flexbox

Theory: Flexbox makes it easier to align and distribute elements in a container.

.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

Practice: Center an element horizontally and vertically using Flexbox.

Quiz: Which CSS property activates flex layout?

Module 6. Responsive Design

Theory: Responsive design adapts your layout to different screen sizes using media queries.

@media (max-width: 600px) {
  body {
    background-color: lightgray;
  }
}

Practice: Change background color for small screens.

Quiz: Which keyword is used to define responsive rules?

Module 7. CSS Transitions & Effects

Theory: Transitions make style changes smooth.

button {
  background: blue;
  transition: background 0.3s;
}
button:hover {
  background: green;
}

Practice: Create a button that changes color smoothly on hover.

Quiz: Which CSS property defines transition duration?
