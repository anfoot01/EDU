Course "React from Scratch"

Module 1. Introduction to React

Theory: React is a JavaScript library for building user interfaces using components and JSX. It is maintained by Facebook and widely used in modern web development.

Setup:

  • Install Node.js from nodejs.org
  • Create a project: npx create-react-app my-app
  • Run the project: cd my-app && npm start
  • Project structure:
    • public/ - static files
    • src/ - source code (components, App.jsx)
    • package.json - dependencies and scripts

Practice: Create your first React app with create-react-app.

Module 2. JSX Basics

Theory: JSX is a syntax extension for JavaScript. It allows you to write HTML-like code inside JavaScript.


function Welcome(props) {
  return <h1>Hello, {props.name}!</h1>;
}

function App() {
  return <Welcome name="Alice" />;
}

Module 3. Components

Theory: Components are reusable UI blocks. There are functional and class components.


function Header() {
  return <header>My App</header>;
}

class Footer extends React.Component {
  render() {
    return <footer>© 2025</footer>;
  }
}

Module 4. Props

Theory: Props (properties) allow you to pass data from parent to child components.


function Greeting(props) {
  return <h1>Hello, {props.name}</h1>;
}

function App() {
  return <Greeting name="Bob" />;
}

Module 5. State

Theory: State allows components to manage data that can change over time. Use the useState hook in functional components. 


import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
}

Practice: Create a counter that increments when a button is clicked.

Quiz: Which hook is used to manage state?

Module 6. Events

Theory: React handles events similarly to HTML, but with camelCase and functions.


function App() {
  function handleClick() {
    alert('Button clicked!');
  }

  return <button onClick={handleClick}>Click me</button>;
}

Module 7. Conditional Rendering

Theory: Render content conditionally using JavaScript expressions inside JSX.


function Greeting({ isLoggedIn }) {
  return <div>
    {isLoggedIn ? 'Welcome back!' : 'Please sign in'}
  </div>;
}

Practice: Display "Welcome" if a variable is true, otherwise "Login".

Module 8. Lists and Keys

Theory: Render arrays using map() and assign a unique key to each element.


function FruitList() {
  const fruits = ['Apple', 'Banana', 'Orange'];
  return <ul>
    {fruits.map((fruit, index) => <li key={index}>{fruit}</li>)}
  </ul>;
}

Practice: Render a list of your favorite movies.

Module 9. Forms

Theory: Handle user input with controlled components.


function NameForm() {
  const [name, setName] = useState('');

  return <form>
    <input value={name} onChange={e => setName(e.target.value)} />
    <p>Hello, {name}</p>
  </form>;
}
