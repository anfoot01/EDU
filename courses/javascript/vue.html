Course "Vue.js from Scratch"

Module 1. Introduction & Setup

Theory: Vue.js is a progressive JavaScript framework for building interactive web interfaces.

Setup:

  • Install Node.js: nodejs.org
  • Install Vue CLI globally: npm install -g @vue/cli
  • Create a project: vue create my-vue-app
  • Run the project: cd my-vue-app && npm run serve
  • Project structure:
    • src/components/ - Vue components
    • App.vue - root component
    • main.js - app entry point
    • package.json - dependencies

Quiz: Which command creates a new Vue project?

Module 2. Components

Theory: Components are reusable building blocks in Vue. Each has a template, script, and optional style.


// Example HelloWorld.vue
<template>
  <h1>Hello {{ name }}!</h1>
</template>

<script>
export default {
  data() {
    return { name: 'Vue' };
  }
}
</script>

<style>
h1 { color: #42b983; }
</style>

Practice: Create a component that displays your name dynamically.

Quiz: Which file extension do Vue components use?

Module 3. Templates & Data Binding

Theory: Vue supports interpolation, v-bind for attributes, v-model for two-way binding, and event binding with v-on.


<input v-model="message">
<p>Message: {{ message }}</p>
<button v-on:click="clearMessage">Clear</button>

Practice: Bind a variable to input and display it in real-time.

Quiz: Which directive binds input data to a variable?

Module 4. Directives

Theory: Directives add reactive behavior to the DOM. Common ones: v-if, v-for, v-show.


<ul>
  <li v-for="item in items" :key="item.id">{{ item.name }}</li>
</ul>

<div v-if="isVisible">Visible content</div>

Practice: Display a list of tasks and conditionally render a message.

Quiz: What directive is used for looping in Vue?

Module 5. Methods & Computed Properties

Theory: Methods define functions, computed properties automatically update based on dependencies.


methods: {
  greet() {
    alert('Hello!');
  }
},
computed: {
  reversedMessage() {
    return this.message.split('').reverse().join('');
  }
}

Practice: Create a button that shows an alert and a computed property that reverses input text.

Quiz: What is reactive and auto-updates when data changes?

Module 6. Components Communication

Theory: Parent-to-child via props, child-to-parent via events using $emit.


// Parent.vue
<Child :message="parentMsg" @notify="handleNotify"></Child>

Practice: Pass a message to child and handle event back in parent.

Quiz: How do children send data to parent?

Module 7. Vue Router

Theory: Vue Router allows SPA navigation between components.


// router/index.js
const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
];

Practice: Create Home and About pages and navigate between them.

Quiz: Which package provides routing in Vue?

Module 8. Mini Projects

  • Todo App: Add, remove, and toggle tasks using v-for and v-model.
  • User List: Display users from a data array and filter them.
  • SPA Navigation: Use Vue Router to switch between pages.
