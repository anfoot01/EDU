Course "Node.js from Scratch"

Module 1. Introduction & Setup

Theory: Node.js is a JavaScript runtime built on Chrome’s V8 engine for running server-side applications.

  • Install Node.js: nodejs.org
  • Check version: node -v and npm -v
  • Create project folder: mkdir my-node-app && cd my-node-app
  • Initialize project: npm init -y

Quiz: Which command initializes a Node.js project?

Module 2. Hello World Server

Theory: Create a simple HTTP server using Node.js built-in http module.


const http = require('http');

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello World\n');
});

server.listen(3000, () => console.log('Server running on port 3000'));

Practice: Start server and access http://localhost:3000.

Quiz: Which module is used to create a basic server in Node.js?

Module 3. Working with Express.js

Theory: Express.js is a minimal framework for building web applications in Node.js.


npm install express
const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send('Hello from Express!');
});

app.listen(3000, () => console.log('Server running on port 3000'));

Practice: Create a basic Express server with one route.

Quiz: Which method defines a GET route in Express?

Module 4. Middleware & Routing

Theory: Middleware are functions that run before route handlers. Use app.use() to apply middleware.


app.use(express.json());

app.post('/data', (req, res) => {
  console.log(req.body);
  res.send('Data received');
});

Practice: Add middleware to parse JSON and handle POST requests.

Quiz: Which Express function handles JSON body parsing?

Module 5. File System & Modules

Theory: Node.js has fs module to read/write files and supports modular code via require. 


const fs = require('fs');
fs.writeFileSync('message.txt', 'Hello Node!');
const data = fs.readFileSync('message.txt', 'utf8');
console.log(data);

Practice: Write a file and read it back in Node.js.

Quiz: Which module allows file operations?

mongoose.connect('mongodb://localhost:27017/testdb');

Module 7. APIs & JSON

Theory: Node.js can serve JSON APIs using Express routes.


app.get('/api/users', (req, res) => {
  res.json([{ id: 1, name: 'Alice' }, { id: 2, name: 'Bob' }]);
