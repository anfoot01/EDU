Module 1. Introduction to Databases and SQL

Theory: JSQL (Structured Query Language) is the standard language for working with relational databases. It is used to create, read, update, and delete data (CRUD operations).

CREATE DATABASE school;

Practice: Create a database called school.

Quiz: What does SQL stand for?

Module 2. Tables and Data Types

Theory: A database consists of tables. Each table has columns (fields) and rows (records).

Common SQL data types:

  • INT – whole numbers
  • VARCHAR(n) - text of length n
  • DATE - dates
  • BOOLEAN - true/false
CREATE TABLE students (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    age INT
);

Practice: Create a table students with columns: id, name, age.

Quiz: Which keyword is used to define a table?

Theory: Insert new rows into a table with INSERT INTO .

INSERT INTO students (id, name, age) VALUES (1, 'Alice', 20);
INSERT INTO students (id, name, age) VALUES (2, 'Bob', 22);
INSERT INTO students (id, name, age) VALUES (3, 'Charlie', 21);

Module 4. Selecting Data

Theory: Retrieve data using SELECT .

Common SQL data types:

  • SELECT * → all columns
  • SELECT name, age → specific columns
SELECT name, age FROM students;

Practice: Get all students’ names and ages.

Quiz: Which symbol selects all columns?

Module 5. Filtering with WHERE

Theory: Use WHERE to filter results.

Operators: =, >, <, >=, <=, <> (not equal).

SELECT * FROM students
WHERE age > 20;

Practice: Find all students older than 20.

Quiz: Which keyword is used to filter rows?

Module 6. Sorting with ORDER BY

Theory: Sort results with ORDER BY .

  • ASC → ascending (default)
  • DESC → descending
SELECT * FROM students
ORDER BY age ASC;

Practice: List students ordered by age (youngest first).

Quiz: Which clause sorts query results?

Module 7. Limiting Results

Theory: Use LIMIT (MySQL, PostgreSQL) or TOP (SQL Server) to restrict number of rows.

SELECT * FROM students
LIMIT 2;

Practice: Get the first 2 students.

Quiz: Which keyword restricts the number of returned rows?

Module 8. Updating Data

Theory: Change existing data with UPDATE.

Module 9. Deleting Data

Theory: Remove rows with DELETE . Be careful: without WHERE it deletes everything!

DELETE FROM students
WHERE name = 'Bob';

Practice: Delete Bob from the table.

Quiz: Which keyword deletes rows?

Module 10. Aggregate Functions

Theory: Aggregate functions work on multiple rows:

  • COUNT() – number of rows
  • SUM() – total
  • AVG() – average
  • MIN() / MAX() 
SELECT AVG(age) AS average_age FROM students;

Practice: Find the average age of students.

Quiz: Which function gives the number of rows?

Module 11. GROUP BY and HAVING

Theory:

  • GROUP BY groups rows by column values.
  • HAVING filters groups (like WHERE , but for aggregates).
SELECT age, COUNT(*) AS count
FROM students
GROUP BY age;

Practice: Count how many students have each age.

Quiz: Which clause groups rows with the same values?

Module 12. Joins

Theory: Join combines rows from multiple tables:

  • INNER JOIN – matching rows only
  • LEFT JOIN – all from left + matches
  • RIGHT JOIN – all from right + matches
CREATE TABLE courses (
    student_id INT,
    course_name VARCHAR(50)
);

INSERT INTO courses VALUES (1, 'Math'), (3, 'Physics');

SELECT students.name, courses.course_name
FROM students
INNER JOIN courses ON students.id = courses.student_id;

Practice: Join students with a new table courses(student_id, course_name) .

Quiz: Which join keeps only rows with matches in both tables?

Module 13. Subqueries

Theory: A subquery is a query inside another query. 

SELECT name, age
FROM students
WHERE age > (SELECT AVG(age) FROM students);

Practice: Find students older than the average age.

Quiz: What is a query inside another query called?

Module 14. Constraints and Keys

Theory:

  • PRIMARY KEY – unique identifier
  • FOREIGN KEY – reference to another table
  • NOT NULL, UNIQUE, DEFAULT 
ALTER TABLE students
ADD CONSTRAINT pk_students PRIMARY KEY (id);

Practice: Add a PRIMARY KEY to students(id).

Quiz: Which constraint ensures unique row identification?
